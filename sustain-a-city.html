01. Overview

The Project

Sustainacity is an eco-systems simulation game rooted in balancing citizen needs with renewable infrastructure. The goal was to create an engaging, immersive experience that challenges players to manage a sustainable city, combining the fun of city-building with the educational value of environmental awareness.

I focused on rapid iteration tools for tweaking resource loops, narrative beats, and designing UI panels that clearly explain environmental impact to the player.

02. Research & Theory

04. Reflection

Post-Mortem

What Went Well

  • Educational Integration: Successfully gamified complex sustainability concepts without making it feel like a "lecture."
  • Visual Style: The Low Poly aesthetic was distinct, performant, and allowed for rapid asset creation during the game jam timeframe.
  • System Depth: The resource loops created genuine strategic depth, requiring players to think ahead.

Areas for Improvement

  • Onboarding: The initial learning curve was steep. A more gradual tutorial system would help retain non-simulation players.
  • DDA Implementation: While researched, the Dynamic Difficulty Adjustment could be more robust to better tailor the experience to different skill levels.
  • Feedback Loops: More visual cues (e.g., smog appearing, trees dying) would help communicate state changes better than just UI numbers.
