# Phase 4: Accessibility — Buttons & Interactive Elements

## Objective

Remove the inline `onclick` handler from the CTA button, add ARIA labels to interactive elements missing them, and ensure all clickable elements are keyboard-accessible.

## Execution Context

- **Branch:** fix/accessibility-interactive
- **Scope:** `index.html`, `assets/js/main.js`
- **Out of scope:** Sub-page accessibility (they have no inline handlers or interactive JS elements beyond standard links), visual changes

## Context

### Discovery Findings

1. **Inline `onclick` handler** — `index.html:65`: CTA button uses inline `onclick` to scroll to `.about` section. Should be moved to `main.js`.

2. **Missing ARIA labels on contact section links** — `index.html:931-974`: The social platform links (GitHub, LinkedIn, Email) in the contact section have no `aria-label`. The profile social links (lines 124-138) already have `aria-label` — good.

3. **CTA button** — Has no `aria-label` but has visible text "About Me" — accessible as-is, just needs the inline handler removed.

4. **Email contact link** — `index.html:960`: `<a>` with `id="email-contact"` — JS already attaches `addEventListener` for copy-to-clipboard. Has `href="mailto:..."` which is good for fallback. But no `aria-label` to indicate the copy behavior.

5. **Project cards** — `<a>` tags with proper `href`, keyboard accessible. No issues.

6. **Sub-pages** — Only have standard `<a>` navigation links. No interactive elements needing attention.

7. **Dither-effect-example.html:107** — Has a `<button>ENTER</button>` with no `aria-label`, but it has visible text "ENTER" — accessible as-is. Standalone demo page, skip.

### Task 2: Remove inline onclick from CTA button
In `index.html:65`, change:
```html
<button class="cta-btn" onclick="document.querySelector('.about').scrollIntoView({behavior: 'smooth'})">
```
To:
```html
<button class="cta-btn">
```

In `assets/js/main.js`, add event listener for `.cta-btn` inside the `DOMContentLoaded` block:
```js
const ctaBtn = document.querySelector('.cta-btn');
if (ctaBtn) {
  ctaBtn.addEventListener('click', () => {
    document.querySelector('.about').scrollIntoView({ behavior: 'smooth' });
  });
}
```

- [ ] Inline onclick removed from HTML
- [ ] Event listener added in main.js
- [ ] CTA button still scrolls to About section
