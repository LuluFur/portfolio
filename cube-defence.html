Mission Brief

Cube Defence serves as my dedicated testing ground for tower-defence mechanics within the Roblox ecosystem.

Rather than a full commercial release, this project focuses on the "feel" of the genre‚Äîspecifically testing enemy pacing, turret upgrade paths, and the critical UI feedback loops that keep players engaged. My primary objective was to create a system where I could shave friction from the iteration process, allowing me to rapidly balance wave spikes and economy curves without getting bogged down in boilerplate code.

Core Pillars

  • Modular Architecture: Hot-swappable tower behaviors.
  • Wave Authoring: Custom tools for spawn cadence.
  • Feedback Loops: Polished UI for player clarity.
