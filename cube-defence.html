Cube Defence

A Roblox Tower-Defence Playground

Engine Roblox Studio
Language Luau
Focus Systems Design
Tower Defense Wave Management UI/UX

The Goal

Cube Defence serves as my dedicated testing ground for tower-defence mechanics within the Roblox ecosystem. Rather than a full commercial release, this project focuses on the "feel" of the genre—specifically testing enemy pacing, turret upgrade paths, and the critical UI feedback loops that keep players engaged.

My primary objective was to create a system where I could shave friction from the iteration process, allowing me to rapidly balance wave spikes and economy curves without getting bogged down in boilerplate code.

Early development progress showing basic pathfinding and tower placement.

Key Systems

Modular Tower Architecture

I built the tower system using modular scripts with hot-swappable behaviors. This allows for distinct attack types like splash damage, single-target high DPS, and status effects (slowing/freezing) to be mixed and matched easily.

Wave Authoring Tools

Balancing a tower defense game is all about the flow. I developed custom wave authoring tools to finely dial in spawn cadence, create mixed enemy compositions, and stage dramatic boss reveals. This tooling allows for rapid testing of difficulty curves.

UI & Feedback

A huge part of the "juice" in tower defense comes from feedback. I implemented polished player-facing UI for upgrade previews, floating damage numbers, and clear resource flow indicators to ensure the player always understands the state of the battlefield.
