Cube Defence

Cube Defence is my Roblox tower-defence playground for testing enemy pacing, turret upgrades, and UI feedback loops. Each build focuses on shaving friction from iteration so I can rapidly balance wave spikes and economy curves.

Roblox Studio Luau Systems Design

Highlights

  • Modular tower scripts with hot-swappable behaviors for splash, single-target, and status effects.
  • Wave authoring tools to dial in spawn cadence, mixed enemy compositions, and boss reveals.
  • Player-facing UI polish for upgrade previews, damage numbers, and resource flows.
